# Project Domain

## Primary Domain
Workforce Planning Analytics

## Supporting Domains
- Human Resources (HR) Analytics
- Employee Attrition and Retention Analysis
- Organizational Performance Analytics

---

## Domain Description

Workforce Planning Analytics focuses on understanding workforce behavior using historical employee data.  
It helps organizations answer key business questions such as:
- Which employees are more likely to leave?
- Which departments experience higher attrition?
- How do satisfaction, performance, and tenure impact retention?

In this domain, data-driven insights are used to:
- Reduce employee turnover
- Improve employee engagement
- Support long-term workforce planning
- Optimize hiring and retention strategies

This project uses **IBM HR Analytics data** to simulate real-world HR decision-making scenarios.
